# 安装docker

这次主要讲述在linux centos 中安装 docker , 主要还是根据官方网站进行

centos install

首先先删除掉之前的版本

 sudo yum remove docker \
                  docker-client \
                  docker-client-latest \
                  docker-common \
                  docker-latest \
                  docker-latest-logrotate \
                  docker-logrotate \
                  docker-engine
